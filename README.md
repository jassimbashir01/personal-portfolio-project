# Jassim Bashir — Developer Portfolio

A modern, high-performance developer portfolio built with **Next.js 16 (App Router)**, **TypeScript** and **Tailwind CSS**. Designed to showcase full-stack development skills, projects and engineering quality with a clean, responsive UI.

<br/>

🚀 Live Demo: https://www.jassimbashir.com

<br/>

## ✨ Features

- ⚡ Fast and optimized with Next.js App Router
- 🎨 Clean UI built with Tailwind CSS
- 🧭 Global layout system (Navbar + Footer)
- 🧑 Hero section with personal branding
- 📁 Featured projects showcase
- 📬 Contact system with:
  - Email (mailto + gmail web fallback)
  - LinkedIn integration
  - Copy-to-clipboard support
  - Inline validation and success states
- 📱 Fully responsive design (mobile-first)
- ♻️ Reusable component architecture
- 🧩 Clean separation of UI components

<br/>
